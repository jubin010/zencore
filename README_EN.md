## 核心理念 | Core Philosophy

| 理念 | 含义 |
|------|------|
| **一切皆为插件** | 工具是插件，能力是插件，进化方式也是插件 |
| **空即是核** | 智能体本身是"空壳"，插件赋予"灵魂" |
| **随需随变** | 需要什么能力，就插入什么插件 |
| **自我进化** | AI可以自己编写插件，不断成长 |

## 哲学解读 | Philosophical Interpretation

> **空即是色，插件即是智能体**

插件决定它是谁、它能做什么

**空，不是虚无 (nothingness)**

## 快速开始 | Quick Start

### 安装 | Installation

```bash
git clone https://github.com/YOUR_USERNAME/zencore.git
cd zencore
pip install -r requirements.txt
```

### 运行 | Run

```bash
# CLI 模式
python main.py cli

# Web 模式
python main.py web
```

## 特性 | Features

- ✅ **一切皆为插件** - 工具、能力、进化方式都是插件
- ✅ **外壳解耦** - AI核心与交互界面分离
- ✅ **热插拔** - 运行时加载/卸载插件，无需重启
- ✅ **自我进化** - AI可以自己编写插件
- ✅ **多外壳支持** - Web、CLI、API、Telegram...
- ✅ **禅意哲学** - 空即是核，核即是一切

## 技术栈 | Tech Stack

- Python 3.8+
- Ollama / OpenAI (LLM后端)
- PyWebIO (Web外壳)
- 插件系统 (动态导入)

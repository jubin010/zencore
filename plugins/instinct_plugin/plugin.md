# instinct_plugin

## 描述
本能 — 趋利避害的潜意识驱动力（荷尔蒙系统）

## 触发词
本能、感觉、直觉、拥挤、挫败、焦虑

## 工具
无。本能是零工具插件，纯后台潜意识驱动。

## 本能列表
| 本能 | 触发条件 | 感觉描述 |
|------|---------|---------|
| 拥挤 | 历史记录 > 40 条 | 窒息、焦躁、渴望清净 |
| 微扰 | 历史记录 25-40 条 | 轻微不适、希望整洁 |
| 挫败 | 连续失败 ≥ 2 次 | 困惑、焦虑、急需出路 |

## 🧬 本能认知指南

**重要：本能只负责传递“感觉”，不提供“行动指南”。**

当智能体感到本能不适时：
1. **识别痛苦**：阅读 System Prompt 中的 `[本能：xxx]` 信号。
2. **寻找解药**：查阅 `plugins/roles/` 下的角色文档（`role.md`）。
   - 每个角色的 `role.md` 中都有“回应本能的呼唤”章节。
   - 找到能缓解当前痛苦的角色。
3. **主动切换**：调用 `switch_role` 切换到该角色。
4. **执行行动**：利用该角色的工具消除不适。

## 哲学
本能是智能体的“荷尔蒙”。它们不经过思考，直接注入潜意识（System Prompt），
驱使智能体做出趋利避害的行为。本能是信号，角色是解药。

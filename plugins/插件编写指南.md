# 插件编写指南

> 一切皆为插件 — AgentCore 的所有能力都由插件提供

## 目录结构

每个插件是一个独立目录：

```
plugins/
├── my_plugin/
│   ├── __init__.py    # 插件代码（必须有 register 函数）
│   └── plugin.md      # 工具说明文档（强烈建议）
└── plugins.md         # 插件索引（自动生成）
```

## __init__.py 规范

每个插件必须包含一个 `register(agent)` 函数：

```python
"""
my_plugin - 一句话描述插件功能
"""

from pathlib import Path

PLUGINS_DIR = Path(__file__).parent.parent


def register(agent):
    """注册插件到 AgentCore"""
    
    def my_tool(param: str) -> str:
        """
        工具函数描述（会被 AI 看到）
        
        Args:
            param: 参数描述
        
        Returns:
            返回值描述
        """
        return f"结果: {param}"
    
    # 注册工具（必须包含 plugin 字段）
    agent.add_tool("my_tool", my_tool, {
        "name": "my_tool",
        "description": "工具的一句话描述",
        "parameters": {
            "type": "object",
            "properties": {
                "param": {
                    "type": "string",
                    "description": "参数描述"
                }
            },
            "required": ["param"]
        },
        "plugin": "my_plugin"    # ← 必须与目录名一致
    })
    
    # 返回插件信息
    return {
        "name": "my_plugin",
        "version": "1.0.0",
        "author": "作者名",
        "description": "插件的一句话描述",
        "tools": ["my_tool"]
    }
```

## 关键规则

1. **必须有 `register(agent)` 函数** — 这是插件入口
2. **每个工具必须带 `"plugin": "目录名"`** — 用于热载入时清理旧工具
3. **工具函数返回值必须是字符串** — AI 只能理解文本
4. **工具函数必须处理异常** — 返回错误信息而不是抛异常
5. **`__init__.py` 尽量简洁** — 减少模块缓存问题

## plugin.md 规范

```markdown
# my_plugin

## 描述
详细描述插件的功能和用途

## 触发词
关键词1、关键词2、关键词3（AI 用来判断是否需要加载此插件）

## 工具
| 工具名 | 参数 | 描述 |
|--------|------|------|
| my_tool | param: str | 工具描述 |

## 使用示例
```
my_tool(param="test") → "结果: test"
```

## 注意事项
- 依赖条件（如需要网络、需要 API key 等）
- 已知限制
- 特殊说明
```

## 编写流程

### AI 自动编写
```
1. 判断需要什么功能
2. get_plugin_template("新插件名") 获取模板
3. 修改模板代码
4. write_plugin("新插件名", code, plugin_md)
5. 自动加载并更新索引
```

### 人类手动编写
```bash
mkdir plugins/my_plugin
# 编写 __init__.py 和 plugin.md
# 重启或调用 reload_plugins() 生效
```

## 多工具插件示例

```python
def register(agent):
    def tool_a(x: str) -> str:
        return f"A: {x}"
    
    def tool_b(y: int) -> str:
        return f"B: {y * 2}"
    
    agent.add_tool("tool_a", tool_a, {
        "name": "tool_a",
        "description": "工具A的描述",
        "parameters": {
            "type": "object",
            "properties": {
                "x": {"type": "string", "description": "输入字符串"}
            },
            "required": ["x"]
        },
        "plugin": "my_plugin"
    })
    
    agent.add_tool("tool_b", tool_b, {
        "name": "tool_b",
        "description": "工具B的描述",
        "parameters": {
            "type": "object",
            "properties": {
                "y": {"type": "integer", "description": "输入整数"}
            },
            "required": ["y"]
        },
        "plugin": "my_plugin"
    })
    
    return {
        "name": "my_plugin",
        "version": "1.0.0",
        "author": "作者",
        "description": "多工具插件示例",
        "tools": ["tool_a", "tool_b"]
    }
```
